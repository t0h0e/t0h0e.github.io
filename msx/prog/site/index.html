Media Station X портал W.A.L.S

Media Station X (сокращенно MSX) – кроссплатформенное приложение доступное телевизоров LG на платформе webOS. Предназначено для создания настраиваемых мультимедийных страниц.
Портал Web App Launche Sсreen (W.A.L.S) для Media Station X - это экран запуска вэб-приложений на вашем телевизоре без необходимости установки.

Как настроить портал W.A.L.S

Установите и запустите приложение Media Station X.

Откройте настройки (Settings),

выберите пункт "Плейлист" (Start Parameter),

нажмите кнопку "Установка"(Setup),

ввести адрес (без http://): wals09.github.io,

подтвердить.
